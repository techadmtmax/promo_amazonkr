# SPN Landing Page

압축을 푼 뒤 아래 순서로 실행하세요.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 을 열면 됩니다.

## 포함 파일
- `components/SpnLandingPage.jsx`
- `app/page.jsx`
- `app/layout.jsx`
- `app/globals.css`
- `tailwind.config.js`
- `postcss.config.js`
- `package.json`

## 참고
- Next.js + Tailwind CSS 구조로 바로 붙여서 사용할 수 있게 구성했습니다.
- 현재 폼은 UI만 포함되어 있으며, 실제 제출 기능은 아직 연결되지 않았습니다.
